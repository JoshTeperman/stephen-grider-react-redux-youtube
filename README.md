## Useful things in this project
- Youtube Data API
- Setting Application HTTP referer restrictions on Public Youtube API keys to accept localhost:3000 only
- iFrame tags: requesting video from Youtube Api and displaying embedded video
- Semantic UI's built-in grid system
- setting default search parameter

Dependencies:
- using axios v 0.18.1 (install using `npm install axios@0.18.1`)
